# webdev-capstone-
capstone html project
Author: Omesh Verma Project: Online Learning Platform (single-file demo) Based on uploaded assignment 'Capstone Project - Responsive Web Design'

Features implemented (per assignment):

Semantic HTML5 structure: header, nav, main, section, article, aside, footer
Hero/banner with CTA and keyframe animation
Course cards displayed with CSS Grid (responsive)
Testimonials, Pricing, and Contact form with labels & accessibility
CSS variables, Flexbox & Grid, two breakpoints (>=1024px, ~768px, <=600px)
Subtle hover effects, transitions, and an animation
Images use placeholder SVGs with meaningful alt text
Skip to main content link and ARIA attributes
